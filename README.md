# STEP 注塑快速报价助手网站包

## 用途

这个目录是可直接上线的静态网站包，包含：

1. `index.html`
2. `assets/product-preview.png`
3. `assets/case-preview.png`

适合部署到：

1. GitHub Pages
2. Gitee Pages
3. 自有服务器
4. 静态网站托管平台
5. 网盘说明页附件包

## 上线前必须替换

在 `index.html` 中搜索 `________`，替换：

1. 试用版下载链接
2. 微信号
3. 公众号名称或正式网址
4. 闲鱼/淘宝/1688 购买链接

## 推荐部署顺序

1. 先用 GitHub Pages 免费上线。
2. 再把链接放到闲鱼、抖音/视频号私信、公众号文章和 1688 私信里。
3. 如果后续需要正式品牌，再买域名和做备案。

## 边界

页面文案统一定位为“前期快速估算工具”，不要改成“全自动精准报价”。
